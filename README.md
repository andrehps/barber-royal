# Royal Cut Barbershop – Modern Web Landing Page

## 📌 Overview
Royal Cut Barbershop is a **modern, responsive, and minimalistic landing page** designed as a digital presence solution for a barbershop.  
The project showcases services, visual identity, and contact options in a clean and intuitive layout.

The interface is structured into well‑defined sections and follows front‑end best practices, using **HTML5, CSS3, and Bootstrap 5** to ensure responsiveness and consistent visual design across devices.

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- Bootstrap 5  
- Google Fonts  
- Modular component structure  

---

## 📂 Section Structure

### **Hero**
Intro section with headline, subheadline, and a call‑to‑action button.

### **About**
Institutional image + descriptive text + visual metrics (years of experience and number of clients).

### **Services**
Responsive cards displaying available services and pricing.

### **Team**
Presentation of barbers and their specialties.

### **Gallery**
A responsive grid showcasing real barbershop‑style images.

### **Contact**
Address, phone, email, and a WhatsApp booking button.

### **Footer**
Simple footer with copyright information.

---

## 🖼️ Images Used
All images represent real barbershop environments and were used **exclusively for demonstration purposes**:

- Modern barbershop interiors  
- Professionals performing haircuts  
- Grooming and styling areas  
- Decorated and stylized spaces  

These visuals enhance the presentation and create a more immersive experience.
